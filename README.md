## 📖 Описание / Description

**[RU]** **DecoSentryPro** — это инструмент автоматизации инфраструктуры для мониторинга Mesh-сетей TP-Link Deco (10+ узлов). Локальное решение с WPF-интерфейсом, заменяющее облачные аналоги для корпоративного сектора.

**[EN]** **DecoSentryPro** is an infrastructure automation tool for monitoring TP-Link Deco mesh networks (10+ nodes). A standalone WPF-based solution designed to replace cloud-based monitoring for enterprise environments.

> [!IMPORTANT]
> **[RU]** Разработано в рамках системы автоматизации для **ЦССВ №11**.
> **[EN]** Developed as part of the infrastructure automation system for **CSSV #11**.

## ✨ Ключевые возможности / Key Features

*   **⚡ Real-time Monitoring:** 
    *   (RU) Многопоточная проверка доступности узлов.
    *   (EN) Multi-threaded node availability checks.
*   **🏗️ IaC Approach:** 
    *   (RU) Конфигурация всей сети через один `config.json`.
    *   (EN) Configuration-driven architecture via `config.json`.
*   **🎨 Custom UI:** 
    *   (RU) Современный "Neon" дашборд на базе WPF/XAML.
    *   (EN) Modern "Neon" WPF dashboard for high-visibility tracking.
*   **📊 Data Export:** 
    *   (RU) Автоматическая генерация CSV-отчетов для аудита.
    *   (EN) Automated CSV generation for inventory and audit compliance.
